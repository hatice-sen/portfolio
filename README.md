#  Kişisel Portfolyo Sitem | Hatice Şen

Bilgisayar Programcılığı eğitimim boyunca geliştirdiğim projeleri, kullandığım teknolojileri ve teknik yeteneklerimi sergilediğim kişisel portfolyo web sitem.

---

##  Kullanılan Teknolojiler
* **Arayüz Tasarımı & Geliştirme:** HTML5, CSS3, JavaScript (ES6+)
* **Tipografi & Stil:** Inter ve Plus Jakarta Sans fontları, modern ve minimalist UI/UX tasarımı
* **Yayınlama (Hosting):** GitHub Pages

---

##  Öne Çıkan Projelerim

Sitem üzerinde detaylarını sergilediğim başlıca çalışmalarım:
1. **ColorVibe (Mobil & Web):** React Native, TypeScript ve Convex kullanılarak geliştirilen, Polaroid tarzı kart arayüzüne sahip dinamik renk paleti yönetim atölyesi.
2. **To Do List:** Minimalist tasarımlı ve gelişmiş filtreleme özellikli mobil yapılacaklar listesi uygulaması.
3. **Video Game Sales Analysis:** vgsales veri seti üzerinde IQR ile veri temizliği, Seaborn ile pazar analizleri ve Random Forest Regressor kullanılan uçtan uca veri bilimi ve makine öğrenmesi projesi.

---

##  Özgeçmiş (CV)
Güncel özgeçmişime site üzerindeki menüden veya doğrudan proje dizinindeki PDF dosyası üzerinden erişebilirsiniz.

Benimle iletişime geçmek, projeler hakkında soru sormak veya iş birlikleri için portfolyo sitemdeki iletişim formunu kullanabilir ya da GitHub profilim üzerinden bana ulaşabilirsiniz!
